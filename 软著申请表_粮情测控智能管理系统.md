十、软件功能和技术特点
────────────────────────────────────────────────────────────────────────────────
运行的硬件环境：
    客户端设备：     PC电脑（Windows 10/11）
    服务器设备：     Intel服务器或云服务器
    数据库服务器：   PostgreSQL
    MQTT Broker：    支持MQTT协议的消息中间件

开发该软件的操作系统：
    Windows 10、Windows 11

软件开发环境/开发工具：
    开发框架：       Next.js 16.1.4
    前端库：         React 19.2.3 + TypeScript
    UI组件库：       HeroUI 2.8.7
    3D渲染：         Three.js 0.182.0 + React Three Fiber
    状态管理：       Zustand 5.0.10
    数据库ORM：      Prisma 5.22.0
    通信协议：       MQTT 5.14.1
    认证：           NextAuth.js 4.24.13
    样式：           Tailwind CSS 4

该软件的运行平台/操作系统：
    Windows 10、Windows 11
    浏览器：Chrome、Edge、Firefox（最新版本）

软件运行支撑环境/支持软件：
    Node.js 18+
    PostgreSQL 14+
    MQTT Broker（如EMQX、Moquitto）

编程语言：          TypeScript + JavaScript
源程序量：          约4600行

主要功能和技术特点：
    功能：
    1. 用户认证与权限管理：支持用户注册、登录、权限审批，支持管理员和普通用户角色
    2. 粮库管理：实现粮库的增删改查，支持按省份、联系人搜索
    3. 仓房管理：实现仓房的增删改查，支持粮库关联
    4. 粮情数据采集：支持串口采集和网络MQTT采集，实现温度、湿度数据实时采集
    5. 数据可视化：支持3D立体视图和2D平面视图展示粮仓温度分布
    6. 数据查询：支持历史数据查询，按时间范围筛选

    特点：
    1. 采用前后端分离架构，基于Next.js框架开发
    2. 使用Prisma ORM实现数据库操作，支持PostgreSQL
    3. 集成Three.js实现粮仓3D可视化，支持温度热力图显示
    4. 支持MQTT物联网协议，实现与粮情采集设备的通信
    5. 采用JWT认证，支持基于角色的权限控制
    6. 响应式设计，支持PC端浏览器访问
